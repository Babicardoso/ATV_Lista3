# 📘 README - Exercícios de Algoritmos e Fluxogramas

Este repositório contém a resolução de uma lista de exercícios de lógica de programação, desenvolvidos por meio de **algoritmos** e **fluxogramas utilizando o Flowgorithm**.

O objetivo das atividades é praticar conceitos fundamentais como entrada de dados, operações matemáticas, estruturas condicionais e validações.

---

## 📌 Exercícios Desenvolvidos

### 1. Soma de 3 números

Desenvolvimento de um algoritmo e fluxograma que realiza a soma de três números informados pelo usuário.

---

### 2. Volume de um paralelepípedo

Criação de algoritmo e fluxograma para calcular o volume com base na fórmula:

```
Volume = Base * Altura * Largura
```

---

### 3. Comissão de vendedor

Algoritmo e fluxograma que calculam a comissão com base no valor do produto:

* Até 5000 → 5% de comissão
* Acima de 5000 → 10% de comissão

---

### 4. Verificação de maioridade e direção

Programa que verifica se uma pessoa pode dirigir:

* Idade ≥ 18 → Pode dirigir
* Idade < 18 → Não pode dirigir

---

### 5. Validação de senha

Sistema simples de autenticação:

* Senha correta: `acb23`
* Caso contrário → Senha incorreta

---

### 6. Cálculo de média e situação do aluno

Algoritmo que calcula a média de três notas e determina a situação:

* Média ≥ 7 → Aprovado
* 4 ≤ Média < 7 → Recuperação
* Média < 4 → Reprovado

---

### 7. Classificação do tipo de voto

Programa que determina se o voto é opcional ou obrigatório:

* Idade entre 16 e 17 → Opcional
* Idade entre 18 e 70 → Obrigatório
* Idade > 70 → Opcional

---

## 🛠️ Ferramenta Utilizada

* Flowgorithm (para construção dos fluxogramas)
* VisuAlg (para construção dos algoritimos)

Consolidar conhecimentos básicos de lógica de programação, preparando para o desenvolvimento em linguagens estruturadas.
